# Changelog

## [1.0.0] - 2026-02-24
### Added
- Initial release
- 10 slash commands: /top10, /top10solo, /top10solodeaths, /top10solokd, /topdamage, /top10allkills, /mystats, /killsagainst, /info, /ping
- SQLite database with kill and loss tracking
- zKillboard API integration
- EVE ESI API for name resolution
- Automated 4-hour kill sync
- Daily leaderboard post at 11:15 UTC
- systemd service for auto-restart and boot persistence
